## 项目概况

开题时间：2024.11.16

基于C++实现⼀个客⼾端-服务器结构的聊天程序，最终呈现结果类似于微信。
- 客户端：基于QT实现
- 服务端：基于C++的分布式微服务架构+主流后端组建
***
- 服务器微服务个数：7个
- 服务器组件个数：17个
- 业务功能点40+
- 总代码量 1.8w
